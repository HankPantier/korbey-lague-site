## What a finished checklist might contain

- Prior year's 1040 + state return
- W-2s, 1099s (NEC + MISC + DIV + INT + R)
- Mortgage interest (Form 1098)
- Property tax statements
- HSA contribution summary
- Charitable donation receipts
- Childcare expenses + provider EIN
- Estimated tax payments made (federal + state)
- Health insurance coverage proof (1095)
